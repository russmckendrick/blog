Co-written with Nassim Kebbani and Piotr Tylenda. The book is a long-form Kubernetes reference that opens with cluster architecture and the responsibilities of each component, then works through container deployment patterns, Pod scheduling, and autoscaling. Cluster setup chapters cover Amazon EKS, Google GKE, and Azure AKS. Later chapters tackle traffic routing into the cluster and deploying applications like Prometheus and Elasticsearch via Helm charts.

Targets developers and DevOps engineers building production Kubernetes workloads.
