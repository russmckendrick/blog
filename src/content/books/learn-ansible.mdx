An introduction to Ansible written for sysadmins and developers who want to turn manual workflows into repeatable playbooks. The book starts with installation and basic playbook structure, then works through configuring Linux and Windows servers, defining cloud infrastructure as code, and using community roles from Ansible Galaxy. Later chapters cover Ansible Tower and Ansible AWX for centralised execution.

No prior Ansible knowledge expected.
