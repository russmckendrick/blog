Extending Docker is about everything that bolts onto the core Docker engine. The first half covers the plugin model: network plugins, volume plugins, and how to write your own. The second half walks through third-party integrations including Puppet, Ansible, Jenkins, Flocker, Rancher, and Packer, then finishes with chapters on schedulers like Kubernetes and Amazon ECS.

Aimed at developers and sysadmins who already know their way around Docker and want to push it further.
