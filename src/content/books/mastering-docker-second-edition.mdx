The second edition of Mastering Docker, co-written with Scott Gallagher. The 2017 release covered building, managing, and storing images, Docker security, and clustering with Docker Swarm and Compose.

Worth tracking down only if you specifically need the older Docker tooling this edition documents. Most readers should look at the Fourth Edition instead. The book is no longer in print.
