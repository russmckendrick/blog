Co-written with Allan Espinosa, this book sits at the intersection of Docker and performance work. Topics include automating Swarm cluster setup with Chef, monitoring with Prometheus and Grafana, log aggregation with the ELK stack, and benchmarking containers using Apache JMeter. Later chapters cover load balancing with NGINX and using Linux tools like strace and tcpdump to debug containers.

Suitable for developers already running Docker who want to tune their setup for production.
