The fourth and current edition of Mastering Docker. Covers building, storing, and distributing container images, integrating Docker into development workflows, and using its security features. The clustering chapters span Docker Compose, Docker Swarm, and Kubernetes via Minikube, Kind, and MicroK8s, with later sections on launching managed clusters in major public clouds.

Aimed at sysadmins, DevOps engineers, and software architects who already have some Docker experience.
