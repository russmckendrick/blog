This book pairs Kubernetes with the early wave of Function-as-a-Service frameworks. Chapters cover Minikube for local clusters, then walk through running serverless workloads on Kubernetes using Kubeless, Funktion, OpenWhisk, and Fission. Later chapters compare these against managed services like AWS Lambda and Azure Functions, and look at monitoring and security for FaaS deployments.

Aimed at DevOps engineers and cloud architects who want to understand what running your own serverless platform actually involves.
