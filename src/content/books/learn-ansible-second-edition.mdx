A rewrite of the original Learn Ansible from 2018. The second edition keeps the same beginner-friendly arc, installation through to advanced playbooks, but the examples now reflect modern Ansible Galaxy usage, deploying to AWS and Azure rather than local VMs, and running playbooks from CI/CD pipelines using GitHub Actions and Azure DevOps. Includes a chapter on scanning playbooks for security issues.

Aimed at the same audience as the first edition: sysadmins, developers, and infrastructure engineers new to Ansible.
