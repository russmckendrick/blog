A practical introduction to Infrastructure as Code aimed at engineers who already deploy resources manually to AWS or Azure and want to put their setups in version control. The first half compares the trade-offs of different IaC tools, then settles into hands-on chapters on Terraform and Ansible, with deployment chapters for both AWS and Azure.

The book also covers running these tools from CI/CD pipelines using GitHub Actions, plus shorter overviews of Pulumi, AWS CloudFormation, and Azure Bicep.
