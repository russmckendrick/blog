The third edition of Mastering Docker, again co-written with Scott Gallagher. Covers Docker Engine, Machine, Compose, and Swarm, with deeper Kubernetes integration than the second edition reflected the state of orchestration in late 2018.

This edition has since been replaced by the Fourth Edition. The book is no longer in print, but the GitHub examples still work for older Docker versions if needed for legacy reference.
