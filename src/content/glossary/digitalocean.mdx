**DigitalOcean** is a cloud platform best known for simple virtual machines called Droplets. It also offers managed databases, Kubernetes, object storage, load balancers, DNS, and application hosting.

Compared with the giant cloud catalogues, DigitalOcean's appeal is focus. It is a good place to spin up a Linux box, test an idea, run a small service, or learn how the moving parts fit together without starting in the deepest end of enterprise cloud.

Simple does not mean disposable. Keep SSH keys under control, patch the machines, watch the bill, and tear down experiments when they are finished. A cheap forgotten server is still a server.
